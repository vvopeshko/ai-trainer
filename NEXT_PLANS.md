## 📅 Среднесрочные итерации

Детали в [BRD.md #12 — Roadmap](BRD.md) (осталось из старой версии — перенести, когда BRD дополню). Коротко:

### Итерация 2. AI-чат (~1 неделя)
- `utils/llm.js` абстракция над Anthropic SDK
- Интеграция чата в бот: текст → LLM → ответ
- Контекст: последние N тренировок пользователя
- Критерий готовности: пишу "чем заменить жим?" → получаю разумный ответ

### Итерация 3. База упражнений + генерация программы (~2 недели)
- Скрипт seed: импорт Free Exercise DB, фильтр, дополнение через LLM
- Таблица `Exercise` + страница "Библиотека" в мини-аппе
- Онбординг-анкета в боте
- `services/aiTrainer/generateProgram.js`: профиль → программа на неделю

### Итерация 4. Фото тренажёра ⭐ (~1 неделя)
- Хэндлер Telegraf на фото в личке
- `llm.vision(image, prompt)` → структурированный JSON
- Inline-клавиатура результата: "начать упражнение" / "альтернативы" / "открыть в мини-аппе"
- Логирование неудачных распознаваний для улучшения промпта

### Итерация 5. Прогресс и визуализация (~1–2 недели)
- Графики в мини-аппе: тоннаж, 1RM по базовым, частота (Recharts)
- Карта прокачки мышц за неделю (SVG-схема тела + подсветка)
- Еженедельная сводка от бота (шедулер)

### Итерация 6. Тесты на друзьях
- 5–20 пользователей, сбор фидбека
- Прайотить по "реально болит"

### Итерация 7. Коммерческая версия
- Подписка + триал (Telegram Stars)
- Лендинг, маркетинг
- Демо-режим (как в daily balancer — client-side)
